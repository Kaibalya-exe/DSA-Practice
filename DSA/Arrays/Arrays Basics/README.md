# Array Basics Practice

## Overview

This folder contains my practice exercises while learning the fundamentals of arrays in C++ and Data Structures & Algorithms.

The goal of these exercises was to understand:

* Array traversal
* Finding minimum and maximum values
* Storing index positions
* Linear search
* Basic problem-solving approaches

---

## Exercises Completed

### 1. Finding Smallest and Largest Elements

**Objective:**

Find the smallest and largest element in an array and return both their values and index positions.

### Concepts Used

* Arrays
* Loops
* Conditional statements
* Variable tracking
* Index management

### Time Complexity

```text
O(n)
```

### Space Complexity

```text
O(1)
```

### 2. Linear Search

**Objective:**

Find a target element inside an array and return its index position.

### Concepts Used

* Array traversal

### Time Complexity

### Space Complexity

