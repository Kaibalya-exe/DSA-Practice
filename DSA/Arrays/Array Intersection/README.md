# Intersection of Two Arrays

## Problem Statement

Given two arrays, print all common elements.

### Example

Input:

```text
arr1 = {1,2,3,4,5}
arr2 = {6,7,3,1}
```

Output:

```text
1 3
```

---

## My Approach

I used a brute-force approach.

For every element in the first array:
- Traverse the second array.
- Check whether a common element exists.
- Print the common element.
- Store the common element in a separate array.

---

## Concepts Practiced

- Nested loops

---

## Complexity Analysis

### Time Complexity

```text
O(n*m)
```

### Space Complexity

```text
O(k)
```

where k is the number of common elements.
