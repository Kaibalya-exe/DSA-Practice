# Pair Sum

## Problem Statement

Given an array and a target value,
find two numbers whose sum equals
the target.

---

## Approach 1: Brute Force

### Algorithm

- Traverse every element.
- Compare with every other element.
- Check if the pair sum equals target.

### Complexity

Time Complexity:

```text
O(n²)
```

Space Complexity:

```text
O(1)
```

---

## Approach 2: Two Pointer Technique

### Algorithm

- Place one pointer at the beginning.
- Place another pointer at the end.
- Compare the sum with target.
- Move pointers accordingly.

### Complexity

Time Complexity:

```text
O(n)
```

Space Complexity:

```text
O(1)
```
